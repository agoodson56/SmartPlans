1. Overview

SmartPlans is an AI-powered construction document analysis platform built by 3D Technology Services Inc. It uses Google Gemini AI to scan your construction blueprints, count devices and infrastructure components, estimate materials and labor costs, and generate professional client proposals — all in minutes instead of days.

Key Capabilities

FeatureDescription
📄 Blueprint AnalysisAI scans uploaded drawing pages to identify devices, symbols, and infrastructure
🔢 Symbol CountingAutomated counting of outlets, cameras, access points, panels, and more
💰 Cost EstimationMaterial pricing and labor hours calculated using industry-standard databases
📋 Proposal GenerationOne-click professional proposal document in Microsoft Word format
📧 RFI GenerationAI identifies gaps in drawings and generates industry-standard RFIs
📊 Multi-Format ExportJSON, Excel, Markdown, and Bill of Materials exports
🏗️ SmartPM IntegrationDirect handoff to project management for installation tracking

Supported Disciplines

  • Structured Cabling — Cat6/6A, fiber optic, patch panels, MDF/IDF rooms
  • CCTV — IP cameras, NVRs, video management systems
  • Access Control — Card readers, controllers, electric locks, REX devices
  • Audio Visual — Displays, speakers, amplifiers, control systems
  • Intrusion Detection — Motion sensors, door contacts, keypads, alarm panels
  • Fire Alarm — Smoke detectors, pull stations, notification appliances, FACP

2. Getting Started

SmartPlans runs entirely in your web browser. No installation is required.

Access URL

https://smartplans-4g5.pages.dev/

Requirements

  • Modern web browser (Chrome, Edge, Firefox, Safari)
  • Construction drawings in PDF or image format (JPG, PNG)
  • Internet connection for AI analysis
💡 Tip: PDF files work best. The AI can extract both visual content and text layers from PDFs for maximum accuracy.

3. Step 1 — Project Setup

The first screen asks you to describe your project. The more detail you provide, the more accurate the AI analysis will be.

FieldRequiredDescription
Project Name✅ YesA name to identify this analysis (e.g., "Sunrise Medical Phase 2")
Prepared For (GC / Agency)❌ OptionalThe General Contractor, agency, or client this proposal is addressed to. Appears on the proposal cover page.
Project Type✅ YesNew Construction, Renovation/Retrofit, or Tenant Improvement. This affects how the AI interprets existing vs. new work.
Disciplines✅ YesSelect which trades to focus on. Focusing on specific trades reduces false positives from similar-looking symbols across disciplines.
File Format✅ YesPDF Blueprints or Scanned Images. This tells the AI which analysis pipeline to use.
Specific Items to Count❌ OptionalEnter specific counting instructions for higher accuracy (e.g., "Count all duplex receptacles on sheets E-101 through E-105").
Project Location❌ OptionalCity, State — used for prevailing wage lookups and travel cost calculations.
Pricing Configuration❌ OptionalChoose pricing tier (Budget/Mid/Premium), prevailing wage type, regional multiplier, and work shift.
⚠️ Important: Targeted counting requests (e.g., "Count all Cat6 outlets on sheets E-101 through E-105") are significantly more accurate than broad requests like "Count everything."

4. Step 2 — Upload Documents

Upload your construction drawings for AI analysis.

Supported Formats

  • PDF — Preferred. Supports multi-page documents. Text layers are extracted for AI context.
  • Images — JPG, PNG, WebP. Each image is treated as one drawing page.

Upload Process

  1. Click "Choose Files" or drag-and-drop files into the upload zone
  2. For PDFs, each page is automatically split into individual sheets
  3. Preview thumbnails appear showing each page that will be analyzed
  4. You can remove unwanted pages before starting analysis
  5. Click Next to begin the AI analysis
💡 Pro Tip: Remove title blocks, general notes pages, and specification pages before analysis. Only include actual drawing sheets with device symbols for best results.

5. Step 3 — AI Analysis

SmartPlans uses a multi-wave AI analysis pipeline to process your drawings:

WaveNameWhat It Does
Wave 0Legend DecoderReads the drawing legend to build a symbol-to-device mapping
Wave 1Drawing AnalysisScans each sheet to count and locate every device
Wave 2InfrastructureIdentifies MDF/IDF/TR room locations and cable pathways
Wave 3Specification ReviewExtracts text specs from PDF for product/standard requirements
FinalMaster EstimatorSynthesizes all data into a complete priced estimate

The analysis typically takes 2–5 minutes depending on the number of pages. A real-time progress display shows which wave is currently processing.

ℹ️ Note: You can watch the AI "thinking" in real-time. Each brain reports its progress and any issues encountered.

6. Step 4 — Results & Review

After analysis completes, you land on the Estimate Complete page showing:

Confidence Score

A 0–100% confidence ring indicating how accurate the AI estimates are, with traffic light indicators (🟢 85%+, 🟡 70-84%, 🔴 below 70%).

Grand Total

The bottom-line total cost including materials, labor, and applied markups.

Analysis Breakdown

  • Material Summary — Every counted item with quantity, unit cost, and extended cost
  • Labor Summary — Labor hours by task type with labor rates applied
  • Infrastructure — MDF/IDF room details with equipment and cabling
  • Location Analysis — Where devices are located across the building

8. Professional Proposal

Click "Generate Professional Proposal" to create a Fortune 500-grade client proposal document.

What's Included

  • Cover Page — 3D TSI branding, project name, prepared for, date
  • Executive Summary — AI-generated scope overview
  • Detailed Pricing — Material and labor breakdown tables
  • Scope of Work — What's included in the estimate
  • Terms & Conditions — Standard 3D TSI terms

The proposal downloads as a Microsoft Word (.docx) file ready for client delivery.

💡 Tip: Fill in the "Prepared For" field during project setup to have the GC/Agency name appear correctly on the proposal cover page.

9. RFI Generation

SmartPlans displays AI-generated Requests for Information (RFIs) — common questions that should be clarified before starting work.

How RFIs Work

  • RFIs are generated based on your selected disciplines
  • Each RFI shows a checkbox — select the ones relevant to your project
  • Click "Export Selected RFIs" to download a text file
  • Selected RFIs are also included in the JSON export for SmartPM import

10. Saved Estimates

Click the 📂 Saved Estimates button in the top-right header to view previously saved analyses.

How Saving Works

  • Estimates are automatically saved when the AI analysis completes
  • Each saved estimate records the project name, date, export data, and all settings
  • Click Load on any saved estimate to restore it
  • All settings, analysis results, and export data are restored
ℹ️ Saved estimates are stored in Cloudflare's cloud database and persist across devices and sessions.

11. SmartPM Integration

After completing your estimate, click the "🏗️ Open SmartPM — Project Manager" button in the Export panel to launch SmartPM in a new tab.

To Import Your Estimate into SmartPM

  1. In SmartPlans, click "JSON — PM App Import" to download the export file
  2. Click "Open SmartPM" to open SmartPM in a new tab
  3. In SmartPM, log in and click "📦 Import from SmartPlans"
  4. Select the JSON file you downloaded
  5. Review the import preview and click "Import Project"

SmartPM will create a fully populated project with locked budgets, infrastructure locations, equipment, cable runs, and RFIs ready to track.

12. Troubleshooting

IssueSolution
AI analysis fails or times outTry reducing the number of pages. Remove non-drawing pages (title blocks, specs).
Low confidence scoreUse "Specific items to count" field with clear instructions. Ensure drawings are high-resolution.
Missing devices in countSelect the correct disciplines. The AI filters by trade to avoid cross-discipline false positives.
PDF won't uploadEnsure the PDF is under 50MB. Try splitting large PDFs into smaller batches.
Proposal formatting issuesThe proposal downloads as .docx — open in Microsoft Word for best results.
Export button not workingCheck your browser's popup blocker. Some exports open new windows.
