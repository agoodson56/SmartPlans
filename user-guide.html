1. Overview

SmartPlans is an AI-powered construction document analysis platform built by 3D Technology Services Inc. It uses a 27-brain AI engine to scan your construction blueprints, count devices and infrastructure components, estimate materials and labor costs, generate professional client proposals, and manage the full bid lifecycle — all in minutes instead of days.

2. Getting Started

SmartPlans runs entirely in your web browser. No installation is required.

Access URL

https://smartplans-4g5.pages.dev/

Requirements

  • Modern web browser (Chrome, Edge, Firefox, Safari)
  • Construction drawings in PDF or image format (JPG, PNG)
  • Internet connection for AI analysis
Tip: PDF files work best. The AI can extract both visual content and text layers from PDFs for maximum accuracy.

3. Seven-Step Workflow

SmartPlans guides you through seven steps from project setup to a finished estimate with full exports.

Step 1 — Project Setup

The first screen asks you to describe your project. The more detail you provide, the more accurate the AI analysis will be.

FieldRequiredDescription
Project NameYesA name to identify this analysis (e.g., "Sunrise Medical Phase 2")
Prepared For (GC / Agency)OptionalThe General Contractor, agency, or client this proposal is addressed to. Appears on the proposal cover page.
Project TypeYesNew Construction, Renovation/Retrofit, or Tenant Improvement. Affects how the AI interprets existing vs. new work.
DisciplinesYesSelect which trades to focus on. Focusing on specific trades reduces false positives from similar-looking symbols across disciplines.
File FormatYesPDF Blueprints or Scanned Images. Tells the AI which analysis pipeline to use.
Specific Items to CountOptionalEnter specific counting instructions for higher accuracy (e.g., "Count all duplex receptacles on sheets E-101 through E-105").
Project LocationOptionalCity, State — used for prevailing wage lookups and travel cost calculations.
Pricing ConfigurationOptionalChoose pricing tier (Budget / Mid / Premium), prevailing wage type, regional multiplier, work shift, travel settings, and labor/material markup percentages.
Important: Targeted counting requests (e.g., "Count all Cat6 outlets on sheets E-101 through E-105") are significantly more accurate than broad requests like "Count everything."

Step 2 — Upload Symbol Legend

Upload the symbol legend page from your drawing set. The AI uses this to build a symbol-to-device mapping before scanning the actual floor plans.

  • Upload the legend page as a PDF page or image
  • The AI decodes every symbol, abbreviation, and device designation
  • This mapping is used by all downstream analysis brains for accurate identification
Tip: If your drawings do not include a separate legend page, the AI will attempt to identify symbols from context, but accuracy improves significantly when a legend is provided.

Step 3 — Upload Floor Plans

Upload the actual drawing sheets that contain device symbols and layout information.

Supported Formats

  • PDF — Preferred. Supports multi-page documents. Text layers are extracted for AI context.
  • Images — JPG, PNG, WebP. Each image is treated as one drawing page.

Upload Process

  1. Click "Choose Files" or drag-and-drop files into the upload zone
  2. For PDFs, each page is automatically split into individual sheets
  3. Preview thumbnails appear showing each page that will be analyzed
  4. Remove unwanted pages (title blocks, general notes) before continuing
Pro Tip: Remove title blocks, general notes pages, and specification pages before analysis. Only include actual drawing sheets with device symbols for best results.

Step 4 — Upload Specifications

Upload specification documents so the AI can perform code compliance checking and extract product/standard requirements.

  • Supports PDF spec documents
  • The Spec Cross-Reference brain matches spec sections against counted items
  • Compliance issues are flagged in the final analysis

Step 5 — Addenda & Supplemental

Upload any change orders, addenda, or supplemental instructions that modify the base drawings or specifications.

  • Addenda are processed after the base drawings to layer in changes
  • The AI adjusts counts, pricing, and scope based on addendum content
  • Multiple addenda can be uploaded and are applied in order

Step 6 — Review Before Analysis

Review all uploaded documents and confirm your project settings before launching the AI analysis.

  • Verify all uploaded files (legend, floor plans, specs, addenda)
  • Confirm project settings, disciplines, and pricing configuration
  • Remove or re-upload any incorrect files
  • Click Start Analysis to launch the 27-brain AI engine
Note: The analysis typically takes 2 to 5 minutes depending on the number of pages. A real-time progress display shows which brain is currently processing.

Step 7 — Estimate Complete (Results)

After analysis completes, you land on the Estimate Complete page. See the Results Page section below for full details on everything available.

6. Supplier Quote Workflow NEW in v5.0

Send your Bill of Materials directly to suppliers, then import their pricing back into SmartPlans for comparison.

Export BOM to Suppliers

  • Export the BOM as an Excel or CSV file formatted for supplier quoting
  • The export includes item descriptions, quantities, and manufacturer part numbers
  • Send the file to one or more suppliers via email

Import Supplier Pricing

  • When suppliers return their quotes, import the completed spreadsheet back into SmartPlans
  • SmartPlans maps supplier line items back to your BOM
  • Supplier pricing replaces or supplements AI-estimated pricing

Track & Compare

  • Track supplier quote status: Sent, Received, Pending
  • Compare supplier pricing against AI estimates side-by-side
  • Choose the best pricing per line item or per supplier
Tip: Use the supplier quote workflow for high-value line items where real-world pricing can significantly improve estimate accuracy.

17. SmartPM Integration

SmartPlans exports are designed for direct import into SmartPM for project management and installation tracking.

Import Workflow

  1. In SmartPlans, click "JSON — PM App Import" to download the export file
  2. Click "Open SmartPM" to open SmartPM in a new tab
  3. In SmartPM, log in and click "Import from SmartPlans"
  4. Select the JSON file you downloaded
  5. Review the import preview and click "Import Project"

What Gets Imported

  • Schedule of Values (SOV) — Budget line items with locked costs
  • Work Breakdown Structure (WBS) — Task hierarchy by discipline and location
  • Infrastructure — MDF/IDF/TR rooms, cable pathways, and equipment
  • RFIs — Selected RFIs are imported ready for tracking

Re-Import to Update

If you revise the estimate in SmartPlans, re-export the JSON and import it again into SmartPM. SmartPM detects the existing project and offers to update it rather than creating a duplicate.

18. Travel & Per Diem

How to Enable

On Step 1 (Project Setup), expand the "Travel & Per Diem" section and check "Include travel costs in estimate." The travel cost summary updates in real-time as you enter values.

The Pricing & Rate Configuration panel controls how SmartPlans prices materials, calculates labor costs, and applies markups to generate your bid price. These settings directly affect the grand total on your estimate.

Material Pricing Tier

Sets the base unit cost for all 200+ items in the built-in pricing database. The AI uses these unit costs when building the Bill of Materials.

TierWhen to UseCat 6A PlenumIP Dome CameraFACP (Medium)

Mid-Range (default)Standard specifications, name-brand products, most commercial projects$0.32/ft$380$5,000
PremiumHigh-end specified products, government/institutional, brand-locked specs$0.48/ft$720$9,500
Note: The tier affects base cost only. Markups are applied separately on top of the tier price.

Regional Cost Multiplier

Adjusts all material costs up or down based on geographic market conditions. Materials cost more in high-cost-of-living areas (Bay Area, NYC) and less in rural/low-cost regions.

Northeast (NYC, Boston)1.35×$13,500

Labor Rates ($/hr)

Hourly base rates for each labor classification. These are the raw hourly wages before burden is applied. The AI uses the appropriate classification for each task (e.g., cable pulling uses Journeyman rate, system programming uses Programmer rate).

ClassificationRoleTypical Range
Journeyman TechStandard installation technician — cable pulling, device mounting, terminations$45-$85/hr
Lead TechSenior technician — supervises small crews, handles complex tasks$55-$95/hr
ForemanOn-site supervisor — coordinates work, manages daily activities (often Lead + 10%)$60-$110/hr
ApprenticeEntry-level — assists journeymen, handles simple tasks (typically 55% of journeyman)$25-$50/hr

Important: If you selected a prevailing wage type (Davis-Bacon or State DIR), these rates are automatically overridden with the legally required prevailing wage rates for your county/state. Your manual entries are used only when "None" is selected.

Labor Burden

The burden rate represents the employer's additional cost on top of the base hourly wage. It covers legally required taxes and insurance that you must pay for every labor hour.

ComponentTypical %What It Covers
FICA (Social Security + Medicare)7.65%Federal payroll tax (employer's matching share)
FUTA + SUTA (Unemployment)2-6%Federal and state unemployment insurance
Workers' Compensation5-12%Injury insurance (rate depends on trade classification)
General Liability Insurance3-5%Commercial general liability (CGL) allocation per labor hour
Health & Welfare8-15%Medical, dental, vision benefits (if offered)
Pension / 401k Match0-6%Retirement contributions (union projects often 8-12%)
TOTAL BURDEN30-45%Default is 35% — adjust based on your company's actual costs

Formula: Loaded labor rate = Base rate × (1 + Burden%). For example: $68.90/hr journeyman × 1.35 burden = $93.02/hr loaded rate.

Markup / Margin Percentages

Markups are applied to your cost to determine the sell price for the client. This is where your profit comes from.

Markup TypeApplied ToDefaultWhat It Means
Material MarkupAll material and equipment costs50%$100 cost → $150 sell price ($50 gross profit)
Labor MarkupAll labor hours (after burden)50%$100 loaded labor → $150 sell price
Equipment MarkupRental equipment (lifts, tools, vehicles)15%$1,000 rental → $1,150 sell price
Subcontractor MarkupAll subcontractor costs (civil, electrical, fire alarm)10%$50,000 sub cost → $55,000 sell price

How the Final Bid Price is Calculated

The Financial Engine brain combines all of these settings into your grand total:

  1. Raw material cost = AI quantities × tier unit prices × regional multiplier
  2. Material sell price = Raw cost × (1 + Material Markup %)
  3. Raw labor cost = AI labor hours × hourly rate × (1 + Burden %)
  4. Labor sell price = Loaded cost × (1 + Labor Markup %)
  5. Equipment sell = Equipment cost × (1 + Equipment Markup %)
  6. Subcontractor sell = Sub cost × (1 + Sub Markup %)
  7. Direct cost subtotal = Material sell + Labor sell + Equipment sell + Sub sell + Travel
  8. G&A overhead = Subtotal × 15% (general and administrative)
  9. Profit = (Subtotal + G&A) × 10%
  10. Contingency = Total × 10% (risk buffer for unknowns)
  11. Grand Total (Bid Price) = Everything above combined
Pro Tip: Use the Bid Strategy feature on the results page to apply different markup percentages per category based on your confidence level — higher markup on uncertain items, lower on well-known costs.

20. Regional Pricing

SmartPlans includes detailed regional pricing data to ensure estimates reflect local labor and material costs.

California County-Level Prevailing Wages

  • All 58 California counties are supported with DIR and Davis-Bacon prevailing wage rates
  • Rates are selected automatically when you enter a California project location

Additional State Coverage

  • 19 additional states with metro-level wage data
  • Rates are matched to the nearest metro area based on project location

Regional Cost Multipliers

  • Multipliers range from 0.8x (low-cost regions) to 1.4x (high-cost regions)
  • Applied to both material and labor costs
  • Can be overridden manually in the pricing configuration

Blended Crew Rates

Labor rates are calculated as blended crew rates that account for journeyman/apprentice ratios, benefits, and burden based on the selected prevailing wage classification.

20. Troubleshooting

IssueSolution
AI analysis fails or times outTry reducing the number of pages. Remove non-drawing pages (title blocks, specs).
Low confidence scoreUse the "Specific items to count" field with clear instructions. Ensure drawings are high-resolution. Upload a symbol legend.
Missing devices in countSelect the correct disciplines. The AI filters by trade to avoid cross-discipline false positives.
PDF won't uploadEnsure the PDF is under 50MB. Try splitting large PDFs into smaller batches.
Proposal formatting issuesThe proposal downloads as .docx — open in Microsoft Word for best results.
Export button not workingCheck your browser's popup blocker. Some exports open new windows.
Numbers don't match between proposal and web appFixed in v5.0. If you are running an older cached version, clear your browser cache and reload.
Supplier quote import failsEnsure the Excel file format matches the SmartPlans supplier quote template. Column headers must not be modified.
Rate library not applyingFuzzy matching requires a partial name match. Ensure saved rate names overlap with BOM item names. Check the match threshold in settings.
